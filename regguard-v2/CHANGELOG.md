# Changelog

## [0.2.3] - 2026-04-08

### Added

- **GitHub issue templates**: Bug report and feature request templates
- **Contact info**: Email support (virgiladoleyine@gmail.com)

- README with contributing guidelines and feedback links
- README with changelog reference

### Added

### Fixed

### Added
- Initial release
- Perplexity integration for regulation lookup
- Grok integration for trending regulations
- Claude integration for code analysis and fixes
- Sidebar with compliance findings
- Line-level diagnostics with quick fixes
- Regulation caching (30 minutes)
- Auto-scan on save option
